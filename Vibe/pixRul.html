Пиксельная рулетка ▼

Инструкция:

1. Загрузите изображение с помощью кнопки ниже

2. Введите базовую длину в поле "Базовая длина"

3. Выберите цвет линии

4. Кликните по изображению, чтобы установить первую точку

5. Кликните второй раз, чтобы установить вторую точку и создать линию

6. Повторите для создания дополнительных линий

7. Правый клик на линии или на числе с длиной для удаления
