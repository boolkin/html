Пиксельная рулетка с множественными линиями

Инструкция:

1. Загрузите изображение с помощью кнопки ниже

2. Выберите цвет линии

3. Кликните по изображению, чтобы установить первую точку

4. Кликните второй раз, чтобы установить вторую точку и создать линию

5. Повторите для создания дополнительных линий

6. Правый клик на линии для удаления
